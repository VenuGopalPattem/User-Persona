# Hacker News Clone - Basic User Stories

## Overview

This document contains basic user stories for a simple Hacker News Clone, focusing only on core functionality.

---

## User Roles

* Visitor (not logged in)
* Registered User

---

## Visitor Stories

* As a visitor, I want to view a list of posts so that I can see what content is available.
* As a visitor, I want to open a post so that I can read its details.
* As a visitor, I want to view comments on a post so that I can follow discussions.
* As a visitor, I want to register an account so that I can participate in the platform.

---

## Registered User Stories

### Authentication

* As a user, I want to log in so that I can access my account.
* As a user, I want to log out so that my account remains secure.

### Posts

* As a user, I want to submit a post so that I can share content.
* As a user, I want to view my submitted posts so that I can track my activity.

### Voting

* As a user, I want to upvote posts so that I can support useful content.

### Comments

* As a user, I want to add a comment to a post so that I can share my thoughts.
* As a user, I want to view comments on posts so that I can engage with discussions.

---

## Notes

* These stories represent only the core features required for a basic version.
* Additional features such as editing, deleting, and advanced filtering can be added later.
